myCar實作範例

自建Fabric節點安裝智慧合約與啟動應用程式語法指令

Note: 以下指令適用於已建立Hyperledger Fabric Network與dev-channel環境下執行。

  • 安裝智慧合約
  • 啟動應用程式

安裝智慧合約

1. 將myCarPrivate-js智慧合約程式下載於專案目錄fabric-dev/chaincode。

2. 啟動自建的Hyperledger Fabric Network。

# 啟動區塊鏈網路
$ cd ~/fabric-dev
$ export FABRIC_CFG_PATH=$PWD
$ export IMAGE_TAG=2.2.2
$ export CA_IMAGE_TAG=1.4.9
$ export SYS_CHANNEL=sys-channel
$ export COMPOSE_PROJECT_NAME=dev
$ docker-compose -f ./docker/docker-compose-test-net.yaml -f ./docker/docker-compose-ca.yaml -f ./docker/docker-compose-couch.yaml up -d 2>&1

3. 打包myCarPrivate-js智慧合約程式。

# 進入Cli Bash
$ docker exec -it cli bash

# 打包智慧合約程式
# 若Fabric Network已安裝過myCarPrivate-js智慧合約，--label可更改智慧合約版本。ex. mycar_1.1
$ peer lifecycle chaincode package mycar.tar.gz --path /opt/gopath/src/github.com/hyperledger/fabric/peer/chaincode/myCarPrivate-js --lang node --label mycar_1.0

4. peer0.org1.example.com節點安裝Chaincode

$ export CORE_PEER_ADDRESS=peer0.org1.example.com:7051

# 安裝智慧合約程式
# 安裝後記下Chaincode code package identifier
# Chaincode code package identifier: mycar_1.0:c243b0af2d1f5ae47ac28c948f9588632759cf2d0f0febc3130d39c3907f04fa
$ peer lifecycle chaincode install mycar.tar.gz

啟動應用程式

1. 將myCarPrivateApp-js應用程式下載於專案目錄fabric-app。

2. 啟動VSCode，並於檔案總管將myCarPrivateApp-js應用程式加入工作區。

3. 在工作區myCarPrivateApp-js應用程式目錄點選右鍵，選擇「在整合性終端機中開啟」。

4. 新增Orderer節點CA與TLSCA憑證。

# 在VSCode下方終端機輸入以下指令
# 新增Orderer排序節點憑證目錄
$ cd ~/fabric-app/myCarPrivateApp-js/
$ mkdir -p ./wallet/peer-cert/ordererOrganizations/example.com/ca ./wallet/peer-cert/ordererOrganizations/example.com/tlsca

# 複製Orderer排序節點憑證
$ cp -i ~/fabric-dev/organizations/ordererOrganizations/example.com/ca/ca.example.com-cert.pem ./wallet/peer-cert/ordererOrganizations/example.com/ca
$ cp -i ~/fabric-dev/organizations/ordererOrganizations/example.com/tlsca/tlsca.example.com-cert.pem ./wallet/peer-cert/ordererOrganizations/example.com/tlsca
